# Introduction

A tiny-funny go framework created base on MVC structure.

## Instllation

Basically run the main.go
`go run main.go`

or use makefile
`make run`

The framework also support hot-reload using fresh, use the below codes.

`make hot`

## Docker

```shell
docker compose build
docker compose up
```
